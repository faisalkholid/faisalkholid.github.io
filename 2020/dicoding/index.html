Jika Aku UI/UX Designer di dicoding, maka sebaiknya beginilah halaman User Profil

Written by Faisal Kholid with ♥ on 12 March 2020 in UI/UX 1 min

Hay nama saya Mohamad Faisal Kholid, kali ini saya mencoba meredesign tampilan website dari dicoding.com yaitu pada bagian User profile.

sebelumnya desain User Profilenya seperti ini :

saya memilih halaman profil untuk diredesign pada halaman www.dicoding.com, saya menggunakan tools Figma untuk membuat tampilan UI baruya saya menghilangkan beberapa element seperti gambar banner yang terlalu besar yang menurut saya tidak terlalu penting ditampilkan dan untuk tata letak dari foto profil saya membuatnya di side bar kanan lengkap dengan “tentang saya” bio dari si pemilik profile, side kanan bersifat tetap tidak bisa di scroll sedangkan sisi kiri adalah side yang bisa di scroll pada tab “tentang saya” dipindah di bawah profil, yang terdapat di side kiri terdapat menu bar academy dll. dengan menggunakan card view dan saya tambahkan untuk progress dibeberapa kelas academy yang dimiliki oleh user. tampilan barunya jadi sepert ini :

Terima kasih telah meluangkan waktu untuk melihat artikel ini, maaf atas salah kata dan sebagainya.

Mungkin bisa jadi inspirasi UI untuk teman-teman yang menjadi UI designer.
